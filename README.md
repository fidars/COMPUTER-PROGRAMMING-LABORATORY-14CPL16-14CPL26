Instructions to run & execute C programs in FEDORA 18:

Open TERMINAL

    Applications -> SystemTools -> Terminal

To create a C file

    $ gedit filename.c

To compile & check the errors

    $ cc filename.c

To Check results/output

    $ ./a.out
